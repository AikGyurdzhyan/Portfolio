# Проект: Разработка API для сервиса Otium

## Цель проекта
-  Реализовать возможность для пользователей стримингового сервиса Otium просматривать список эксклюзивных фильмов и сериалов, доступных только на платформе.
-  Упростить пользователям выбор фильмов и сериалов на платформе Otium за счёт добавления рейтинга кинокритиков и возможности сортировки контента по этому рейтингу.

## Выполненные задачи
- Проанализирована диаграмма компонентов системы в нотации **UML**, отражающая архитектуру Otium. 
- Изучена коллекция **REST API** сервиса «Фильмы»: рассмотрены методы, параметры запросов и структура ответов.  
- На основе анализа существующего API определены требования для нового сервиса.  
- Составлена диаграмма последовательности в нотации **UML**, демонстрирующая взаимодействие пользователя с сервисами "Фильмы" и "Сериалы", с фильтрацией и сортировкой по эксклюзивному контенту, доступных только на платформе Otium.  
- Подготовлены предложения по расширению API: эндпоинты для получения списка эксклюзивного контента с учетом фильтрации и сортировки по рейтингу кинокритиков.

## Учёт бизнес-ограничений
- Новый сервис должен быть интегрирован в существующую архитектуру Otium через слой API.  
- Контент без рейтига должен отображаться в конце списка независимо от выбранного порядка сортировки.
- Обеспечить корректную работу сортировки в сочетании с существующими фильтрами (сначала фильтрация, затем сортировка).  
- Все компоненты должны поддерживать работу в веб-браузере, ПК-приложении, мобильном приложении и приложений для ТВ.
